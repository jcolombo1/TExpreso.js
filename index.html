TExpreso

Liviano y veloz renderizador de contenidos a través de plantillas (templates) similar a Handlebars y Mustache. Adiciona "variables" y otras mejoras para facilitar su uso.

TExpreso

A diferencia de Handlebars y Mustache, cuenta con estructuras "if..else..end", además de auto-path del contexto de datos y bloques auto-condicionales y auto-iterativos (luego los veremos).

Es súmamente simple de usar y al igual que Handlebars permite embeber funciones "helpers" en runtime.

Incluye un plugin de jQuery (jquery.TExpreso.js) que facilita tanto la carga de templates o helpers desde urls remotas, como también su renderización dentro de elementos html del DOM.

Sin mostrar aún los métodos de renderizado, veamos un par de ejemplos de templates y datos para entender su funcionalidad...


Nota: Estos ejemplos están contenidos en el archivo "example.html" (obsérvelos en su explorador).

Ejemplo 1:

Este pequeño template:

  <p>{{saludo}} {{ nombre }} ! </p>

al renderizarlo con el siguiente contexto de datos mediante el javascript:

var datos = { nombre: "Jorge", saludo: "Buenos días" };

el resultado será:

<p>Buenos días Jorge ! </p>

Ejemplo 2: 

<!-- Template -->

<div>
  {{# if persons.length > 0 }}

    <ul>
        {{# persons }}
            <li> <b>{{firstName}} {{lastName}}</b> vive en {{address.city}} </li> 
        {{ end }}  
    </ul>
    <p>Hay {{persons.length}} personas</p>

  {{ else }}
     <p>No hay personas!</p>
  {{ end }}  
</div>

al renderizarlo con el siguiente contexto de datos:

<!-- ... contexto de datos a renderizar: -->
var myContext = {
  persons: [
    { firstName: 'Jorge', lastName: 'Colombo', address: { city: 'Los Polvorines', province: 'BA' }  },
    { firstName: 'Patry', lastName: 'Gutierrez', address: { city: 'Los Polvorines', province: 'BA' }  },
  ]
};

el resultado será:

<div>
  <ul>
      <li> <b>Jorge Colombo</b> vive en Los Polvorines </li> 
      <li> <b>Patry Gutierrez</b> vive en Los Polvorines </li> 
  </ul>
  <p>Hay 2 personas</p>
</div>

Pero, al renderizarlo con este contexto:

var myContext = { persons: [] };

el resultado será entonces:

<div>
  <p>No hay personas!</p>
</div>
